Welcome to SCCM

Configure and manage all aspects of this service as follows:

Security Configuration

Allows you to set up the following security aspects:

  • Create, modify and delete users and groups, and assign roles.
  • Configure the import of users and groupts from your LDAP server.

Users, Groups and Roles

Manage users, groups and roles:

  • Users

    Create, edit or delete users. You can also view users that were imported from your LDAP server. However, you cannot modify those users from this administration client.

  • Groups

    Create, edit or delete groups. You can also view groups that were imported from your LDAP server. However, you cannot modify those groups from this administration client.

  • Roles

    Assign roles to groups and users to grant them different privileges.
