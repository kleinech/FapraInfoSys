# FapraInfoSys
### Requirements:
1. npm ^3.x (test with command: `npm -v`
2. For Content (Users, Groups):
    - entitlement-service started
    - vpn connection to Uni Stuttgart (ldap)
    - (Without entitlement-service => alert messages)

### How to use:
1. install node.js
2. `npm install`
3. `npm start`
